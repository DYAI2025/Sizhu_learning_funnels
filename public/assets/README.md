# Public Assets

Store optimized public images, icons, social preview images, and downloadable non-sensitive assets here.

Requirements:

- record source and license where applicable;
- provide meaningful alternative text in page content;
- keep critical terminology as real text rather than rasterized image text;
- do not commit private customer material, birth data, charts, credentials, or licensed font files;
- use predictable, lowercase, hyphenated filenames.
