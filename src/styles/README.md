# Styles

This workspace owns shared design tokens, typography, layout primitives, component styles, responsive behavior, visible focus states, and reduced-motion behavior.

Rules:

- reuse one Sizhu Atelier design system across all learning pages;
- do not encode meaning through color alone;
- maintain readable line length and mobile layouts;
- isolate page-specific selectors;
- avoid global overrides without regression evidence;
- document external font and asset dependencies.
