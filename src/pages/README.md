# Page Routes

This directory owns canonical learning page composition.

Until a framework or static-site generator is explicitly selected, do not invent framework-specific file names or routing conventions. Use `src/routes.json` as the canonical route inventory and document any implementation mapping in the relevant Jira plan and pull request.

Every page must provide:

- one canonical route;
- one H1;
- semantic section hierarchy;
- breadcrumbs;
- source and limitations section;
- related-learning links;
- explicit CTA target types;
- metadata and structured-data inputs;
- accessibility and analytics verification.
