# Shared Components

Shared components should cover cross-page concerns such as:

- site header and footer;
- breadcrumbs;
- table of contents;
- related-learning cards;
- source and review blocks;
- limitations and evidence notices;
- Wu Xing diagrams and legends;
- product, Etsy, Learn, and Bazodiac CTAs.

Components must remain content-agnostic, keyboard accessible, responsive, and usable without requiring personal data.
